Intona.AI

By Christopher Lock
Oct / 2 / 2023

An AI powered Max/MSP instrument and sound design tool for experimental music.

This project uses RAVE Antoine Caillon and Philippe Esling 
For more info: https://github.com/acids-ircam/RAVE 


Step 1:
Download RAVE from https://github.com/acids-ircam/RAVE

Rave is a Realtime Audio Variational autoEncoder.


Step 2: 
Download nn~ here https://github.com/acids-ircam/nn_tilde


Step 3:
Make sure these packages are in the packages folder of Max/MSP


Step 4:
Unzip the file Models.zip into Documents -> Max 8 -> Packages -> nn_tilde -> externals


Now the patch should be all set to run.
